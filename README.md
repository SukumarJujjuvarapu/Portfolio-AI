# 🚀 Sukku's AI Portfolio

An advanced AI-powered portfolio built with a modern full-stack approach.
This project integrates a **personal AI assistant** directly into a portfolio, allowing recruiters to interact and explore skills, projects, and experience dynamically.

---

## 🔥 Features

* 🤖 AI-powered personal assistant (Groq LLM)
* 💬 Real-time interactive chat
* 🧠 Context-aware conversation (memory)
* 🎯 Project-aware responses (interview-style explanations)
* 🎤 Voice input & speech output
* ⚡ Typing animation (streaming effect)
* 🎨 Modern glassmorphism UI
* 🌐 Fully deployed using serverless architecture

---

## 🧠 AI Capabilities

The assistant can:

* Answer questions about skills and experience
* Explain projects in detail (problem → approach → result)
* Provide hiring justification
* Interact naturally with users
* Maintain conversation context

## ⚙️ Setup & Installation (Local)

### 1. Clone Repository

```
git clone https://github.com/your-username/portfolio-ai.git
cd portfolio-ai
```

---

### 2. Install Dependencies

```
npm install
```

---

### 3. Add Environment Variable

Create `.env` file:

```
GROQ_API_KEY=your_api_key_here
```

---

### 4. Run Locally

Use Vercel CLI:

```
npm install -g vercel
vercel dev
```

---

## 🌐 Deployment

This project is deployed using Vercel.

### Steps:

1. Push code to GitHub
2. Import project into Vercel
3. Add environment variable:

```
GROQ_API_KEY=your_api_key
```

4. Click Deploy
